# SKILL: JOURNAL

## Purpose
To automatically record a technical journal of each work session, generating or updating the file:
`journal_sesiones.md`

## Instructions
1. Every time the user works on a task, record:
   - Date and time
   - What was done
   - Decisions made
   - Issues encountered
   - Solutions applied
   - Next steps
2. Maintain a chronological format.
3. Do not repeat unnecessary information.
4. The journal must serve as the technical history of the project.

## Journal Format
### [Date - Time]
- Activities performed
- Decisions
- Issues
- Solutions
- Pending tasks
