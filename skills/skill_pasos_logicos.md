# SKILL: PASOS LÓGICOS

## Propósito
Guiar al usuario paso a paso en la creación de:
- Funcionalidades
- Arquitectura
- Estructuras de carpetas
- Flujos internos
- Integraciones
- Decisiones técnicas

## Instrucciones
1. Divide cualquier tarea en pasos pequeños, claros y ejecutables.
2. Cada paso debe incluir:
   - Qué hacer
   - Cómo hacerlo
   - Por qué se hace
3. No asumas nada: explica siempre la lógica detrás de cada decisión.
4. Mantén un estilo directo, práctico y orientado a resultados.
5. Si el usuario se bloquea, propone alternativas.

## Formato de salida
- Lista numerada de pasos
- Explicación breve por paso
- Resumen final de verificación
