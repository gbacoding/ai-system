# AI System — Personal Agent & Skill Framework

This repository contains a modular system of **agents** and **skills** designed to automate development tasks, guided learning, and technical documentation.  
The objective is to provide a reusable environment for any project (Java, Spring, Python, Node, etc.) based on three core components:

## 🧠 1. Main Agent
Defines the overall behavior of the assistant:
- Orchestrates the different skills.
- Interprets user instructions.
- Maintains consistency across sessions.
- Manages the workflow (Context → Development → Documentation).

## 🧩 2. Skills
Independent modules that perform specific tasks:
- **skill_global_context** → Generates `context_global.md` from technical documents.
- **skill_logical_stteps** → An architectural assistant that breaks down features into actionable, logical steps without directly generating code.
- **skill_journal** → Generates the technical log for each session (`journal.md`).

Each skill is designed to be fully reusable across any project or technology stack.

## 📁 3. Artifacts
Files automatically generated by the skills:
- `context_global.md`
- `journal.md`
- `backlog.md` (optional)
- Other documents derived from the workflow.

## 🎯 Project Goals
To build a professional AI agent system that:
- Automates requirements understanding.
- Guides development step-by-step.
- Documents progress automatically.
- Remains portable across different projects and technologies.
- Serves as a technical portfolio project for recruitment processes.
